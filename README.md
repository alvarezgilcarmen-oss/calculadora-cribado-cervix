# calculadora-cribado-cervix
Esta calculadora convierte los circuitos de cribado y seguimiento del cáncer de cuello de útero en una experiencia guiada: paso a paso, pregunta a pregunta, hasta llegar a la conducta correcta. Una herramienta diseñada para mejorar la práctica clínica, reducir errores y avanzar hacia un cribado más eficiente y homogéneo.
